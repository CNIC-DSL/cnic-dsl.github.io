招聘信息

开放岗位

我们正在招募有热情的研究员、博士后、博士研究生和硕士研究生加入团队，欢迎联系加入！

我们正在寻找充满热情、才华横溢、并具有坚韧精神的新成员加入我们的团队！

您将有机会参与人工智能领域与各交叉科学领域中的重大挑战，包括但不限于智能体系统、多模态知识建模、人机协同机制，以及AI驱动的科学发现。在这里，您不仅可以深入参与前沿研究问题的定义和理论方法的探索，还将亲手构建原型系统、优化算法性能，并推动真实世界中的应用落地。我们期待您在复杂问题中展现创造力，在科研与工程的交汇处不断突破边界，与我们共同开拓人工智能未来的新范式。

博士后招聘信息

我们热烈欢迎优秀青年学者申请加入计算机网络信息中心开展博士后研究工作。我们都将为您的科研发展提供全力支持，包括但不限于支持申报国家博士后各类科学基金、中科院特别研究助理计划等。欢迎通过邮件与我们联系，附上您的简历、研究背景介绍和资助申请意向，我们期待与志同道合的您一起探索科研前沿。

科研实习生招聘信息

如果您是在读本科、硕士、博士，正在寻找实习或毕业项目，欢迎通过邮件联系。

硕士/博士研究生招生信息

我们欢迎来自全国各高校的优秀本科生或硕士生申请在计算机网络信息中心攻读硕士或博士学位。如果您对人工智能、数据智能、科学计算等方向的前沿研究充满热情，欢迎您申请加入我们的研究团队。

热忱欢迎全国在校本科生报名2026年中国科学院计算机网络信息中心“网络与计算”夏令营，更多详细信息欢迎通过邮件联系。

工程师与运营主管岗位招聘信息

岗位名称：全栈研发工程师（AI平台方向）（1人） 用工方式：项目聘用/劳务派遣 岗位职责： 1. 与算法研发团队紧密合作，负责人工智能模型的上线部署、开发及维护，确保模型在生产环境中高效、稳定运行，负责数据相关的预处理、管理； 2. 与产品团队密切合作，负责科学智能平台的建设工作，包括后端模型管理、API服务及监控系统、平台前端的开发与测试等，以及相关平台的运维工作； 3. 负责项目相关研发文档的整理与撰写工作； 4. 项目研发需要的其他任务。 任职要求： 1. 计算机相关领域本科及以上学历； 2. 熟悉大模型等AI模型部署流程，包括Docker/Kubernetes环境下的模型服务化经验； 3. 熟练掌握Python，熟悉相关AI/ML框架（如PyTorch等）； 4. 熟悉Linux操作系统及常用命令，具备自动化运维经验； 5. 具备全栈开发能力，熟悉数据库、后端开发（如FastAPI、Flask、Spring Boot）及前端（如React/Vue）相关技术； 6. 具备良好的代码习惯、文档撰写、系统设计及问题定位能力。 加分项： 对大模型、智能体等AI前沿技术敏感，有实操经验，能够快速将新技术应用到平台和业务中； 学习能力强、主动性高，具备良好的沟通能力和团队协作精神。 岗位名称：Web前端工程师（AI平台方向）（1人） 用工方式：项目聘用/劳务派遣 岗位职责： 1. 负责Web前端架构设计、页面开发与功能实现，保障代码质量与可维护性； 2. 根据产品和业务需求，完成高质量的用户交互界面（UI）与交互逻辑（UX）开发，持续优化用户体验； 3. 与后端开发人员协作，完成接口对接、数据交互及联调工作； 4. 负责项目相关研发文档的整理与撰写工作； 5. 项目研发需要的其他任务。 任职要求： 1. 计算机相关领域本科及以上学历； 2. 精通 HTML5、CSS3、JavaScript（ES6+），熟悉现代浏览器原理； 3. 熟练掌握至少一种主流前端框架（如Vue.js），具备组件化、模块化开发经验； 4. 熟悉前端工程化工具与流程，如Git、Webpack/Vite、npm/yarn等； 5. 了解常见Web性能优化、安全与兼容性问题； 6. 具备一定UI/前端美工设计能力； 7. 具备良好的代码习惯、文档撰写及问题定位能力。 岗位名称：运营主管（科学数据方向）（1人） 用工方式：项目聘用/劳务派遣 岗位职责： 1. 负责科学数据相关产品的全链路运营统筹，搭建并完善科学数据产品品牌体系，联动内外部资源推动产品影响力提升，助力科学数据资源的价值转化与行业普及； 2. 围绕科学数据相关产品规划并搭建多维度品牌矩阵，如媒体、白皮书、奖项等，负责官方媒体矩阵（网站、公众号、行业社群等）的内容策划和日常运营； 3. 组织策划并落地各类科学数据相关运营活动，包括发布会、研讨会、学术论坛等，负责活动全流程管理，协调内外部资源，保障活动顺利开展，推动产品推广与合作转化； 4. 负责科学数据领域国际、国内标准的统筹运营工作。跟踪行业标准动态，联合技术、研发团队及行业专家，组织标准立项、编制、评审等工作；负责标准发布后的宣贯推广，推动标准在行业内的落地应用与普及。 任职要求： 1. 数据科学、信息管理、传播学等相关专业本科及以上学历，熟悉科学数据领域行业动态、政策法规及发展趋势，了解科学数据产品的特性与运营逻辑，2年以上科学数据、科研信息化、大数据相关行业运营经验； 2. 具备品牌矩阵搭建与内容策划能力，能独立完成白皮书、行业报告等专业内容的统筹，熟悉媒体运营逻辑与奖项申报流程； 3. 了解科学数据领域国际、国内标准体系及制定流程，具备较强的逻辑思维与文档撰写能力；具备良好的英语听说读写能力，可独立完成国际标准对接、英文材料撰写等相关工作。
